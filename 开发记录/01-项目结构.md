# 开发记录 - 01 项目结构

## 1. 技术栈 (Tech Stack)
- **核心引擎:** Three.js (r181)
- **开发语言:** TypeScript
- **构建工具:** Vite
- **运行环境:** Browser (win32 开发环境)
- **自动化测试:** Playwright (用于逻辑自测)

## 2. 目录布局 (Folder Structure)
```text
/need-chain-prototype
├── /开发记录            # 项目开发文档 (当前目录)
├── /src
│   ├── /core           # 核心引擎 (Game.ts 主循环)
│   ├── /entities       # 游戏实体 (Player, Item)
│   ├── /systems        # 功能系统 (AIDirector, Enemy, Vfx, Sound, etc.)
│   ├── main.ts         # 入口文件
│   └── style.css       # 样式与全屏 UI 布局
├── index.html          # HUD 结构与 Canvas 容器
├── diagnose.cjs        # 自动化测试脚本
└── package.json        # 依赖管理
```

## 3. 开发规范
- **VibeCoding 优先:** 优先保证视觉和手感反馈 (Vibe)，逻辑采用 Context 注入模式以便快速重构。
- **动态清理:** 所有动态生成的物体在 `initWorld` 时需打上 `isDynamic` 标签，以便统一清理。
- **坐标系:** 100x100 方形盒子, 范围 [-50, 50]。
