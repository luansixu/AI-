# 开发记录 - 04 数据结构定义

## 1. PlayerState
```typescript
interface PlayerState {
  hp: number;       // 0-100
  maxHp: number;
  temp: number;     // TP, 0-100
  stamina: number;  // SP, 0-100
  greed: number;    // 贪婪值
  alert: number;    // 惊扰值
  holding: Item | null;
}
```

## 2. ItemConfig
```typescript
type ItemType = 'Heavy_Sword' | 'Heat_Source' | 'Tool' | 'Frost_Heart';
interface ItemConfig {
  type: ItemType;
  color: number;
  weight: number;   // 影响玩家速度
}
```

## 3. EnemyData
```typescript
interface EnemyData {
  group: THREE.Group;
  hp: number;
  knockbackVel: THREE.Vector3;
  kind: 'normal' | 'blue_boss';
}
```

## 4. CollisionObject
```typescript
interface CollisionBody {
  pos: THREE.Vector3;
  radius: number;
}
```
