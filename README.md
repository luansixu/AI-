# AI Native 沙盒：需求链（Need-Chain）原型

本原型旨在展示一种由 **AI 地下城主 (AI-DM)** 驱动的次世代沙盒游戏交互范式。

## 🎮 游戏核心思想
我们抛弃了传统沙盒游戏中死板的“KPI 式任务系统”，转而追求 **真实需求驱动 (True Need-Driven)** 的沉浸式体验。游戏世界不再是静态的背景，而是一个具备“逻辑推演”能力的动态进化世界。

## 💡 核心创意：需求链 (Need-Chain)
AI-DM 通过实时监测玩家的内部状态（如体温、体力、负重）与外部行为（如贪婪度、惊扰值），自发地在环境中制造“麻烦”或提供“诱惑”，从而引导玩家设定自己的目标。

**案例演示：**
- **生存施舍：** 当玩家体温过低濒临死亡时，AI 会投放火堆，但火光会瞬间吸引大量怪物（高惊扰值代价）。
- **贪婪惩罚：** 玩家拾取强力的“白色圣剑”会增加贪婪值，导致 AI 派怪围攻。
- **能力门槛：** 为了抵御极寒，玩家必须击败“蓝色冰霜精英”以获取其掉落的“冰霜之心”，改变自身语义属性。

## 🚀 解决的痛点
- **拒绝“通马桶”任务：** 每一项行动都源于真实的生存压力或能力渴望。
- **打破内容边界：** 动态生成的基于逻辑的挑战，让后期体验依然充满变数。
- **活的世界：** 玩家的行为会引发蝴蝶效应，世界会根据你的选择产生连锁反应。

## 🛠️ 技术实现
- **引擎：** Three.js (3D 渲染)
- **架构：** 基于 Context 的模块化系统 (TypeScript)
- **逻辑：** 巫师模式 (Wizard of Oz) 模拟 AI 推演引擎
