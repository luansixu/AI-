# Need-Chain Prototype: AI Native 沙盒生存原型

这是一个基于 **Three.js** 和 **TypeScript** 开发的 AI Native 游戏原型。其核心设计哲学是 **“需求驱动（Need-Chain）”**：游戏中的 AI 导演（Wizard of Oz 模式）会实时感知玩家的生存困境，并通过环境语义（石碑、视觉预兆）进行隐性引导，而非直接派发任务。

## 🌟 核心特色

- **AI-DM 导师化：** 导演系统会分析玩家的库存与状态。例如，当你拥有皮毛却忍受严寒时，AI 会修改地图上的石碑文字，暗示“皮毛可作寒衣”，引导你发现合成公式。
- **双极环境博弈：** 地图划分为“极寒冰原”与“烈焰焦土”。玩家需要通过“交叉探索”来获取通行证（在冰原寻找烈焰区的保护核心）。
- **硬核生存闭环：** 涵盖生命、体力、饱食度、体温四大维度。失温不再是即死，而是真实的生命流失感。
- **草蛇灰线引导：** 移除 UI 引导箭头，采用冲天圣光（出生点光柱）、古朴石碑和环境粒子（雪花/余烬）作为视觉语言。

## 🎮 操作指南

| 按键 | 功能 | 说明 |
| :--- | :--- | :--- |
| **W/A/S/D** | 移动 | 角色会根据朝向自动旋转 |
| **Shift (按住)** | 疾跑 | 消耗体力，提升移动速度 |
| **J** | 攻击 / 采集 | 挥舞圣剑砍树、挖矿、狩猎或御敌 |
| **E** | 交互 / 拾取 | 拾取掉落物、阅读石碑、使用祭坛 |
| **U** | 合成篝火 | 消耗 3 份木材，手搓一份篝火物品 |
| **I** | 合成皮衣 | 消耗 2 份皮毛，制作耐寒装备 |
| **K** | 放置篝火 | 在脚下部署一个救命的热源 |

## 🏹 远征路线（通关目标）

1. **拔剑：** 拾取出生点祭坛旁的白色断刃。
2. **原始积累：** 砍伐树木（木材）并猎杀绵羊（皮毛）。
3. **武装自己：** 按 **U** 合成篝火，按 **I** 制作皮衣。
4. **冰原探险：** 深入东北方（右上）极寒区，击碎冰晶矿，并寻找由 AI 投放的 **“熔火核心”**。
5. **征服烈焰：** 持有核心进入西南方（左下）烈焰区，击碎火矿石。
6. **重铸平衡：** 带回 1 份冰晶与 1 份火矿，回到坐标 (40, 40) 的光柱祭坛，按 **E** 完成重铸。

## 🛠️ 技术栈

- **Engine:** Three.js (r181)
- **Language:** TypeScript
- **Bundler:** Vite
- **Vfx:** Procedural Particle System (Custom)
- **Audio:** Procedural Web Audio API (Placeholder)
